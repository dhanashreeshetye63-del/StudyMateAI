# ЁЯОУ StudyMate AI тАУ Personalized Learning Assistant

## ЁЯУЦ Project Overview

StudyMate AI is an AI-powered learning assistant developed to support **United Nations Sustainable Development Goal (SDG) 4: Quality Education**.

The application helps students by providing instant answers to study-related questions, suggesting learning topics, and offering a simple dashboard to track their learning.

---

## ЁЯМН SDG Selected

**SDG 4 тАУ Quality Education**

Quality education ensures inclusive and equitable learning opportunities for everyone. StudyMate AI aims to make learning more accessible by providing students with an intelligent study assistant.

---

## тЭЧ Problem Statement

Many students struggle to find reliable learning resources and receive instant help while studying. Personalized guidance is often unavailable, especially for beginners. This can reduce motivation and slow down the learning process.

---

## ЁЯТб Proposed Solution

StudyMate AI provides a simple AI-based learning assistant where students can:

- Ask study-related questions
- Receive AI-generated responses
- Follow a daily study plan
- View learning progress
- Access recommended study topics

## ЁЯЪА How to Run

1. Clone the repository

```bash
git clone https://github.com/yourusername/StudyMateAI.git
```

2. Open the project folder

```bash
cd StudyMateAI
```

3. Install dependencies

```bash
pip install -r requirements.txt
```

4. Run the application

```bash
python app.py
```

5. Open your browser

```
http://127.0.0.1:5000
```
